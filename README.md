# Hospital Management System V2 (MAD-II)

A full-stack Hospital Management System built as part of the **Modern Application Development II** course at IIT Madras.

## Project Description

This application enables **Admins**, **Doctors**, and **Patients** to interact with a hospital system based on their roles.

- **Admin** can manage doctors, patients, departments, and appointments.
- **Doctors** can view scheduled appointments, manage availability, and record diagnoses/prescriptions.
- **Patients** can register, book/cancel appointments, and view their treatment history.

## 🚀 Quick Start

1. **Install Dependencies:**
   ```powershell
   pip install -r requirements.txt
   ```

2. **Run the Application:**
   ```powershell
   python app.py
   ```

3. **Access the Application:**
   - Open your browser and go to: `http://127.0.0.1:5000`
